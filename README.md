#  Movie Rating System

A Python + SQLite-based console application to manage movies, users, and ratings.  
This project demonstrates database management, CRUD operations, and user interaction in a simple rating system.

---

##  Features
- Add, view, and delete movies
- Add users with contact information
- Rate movies (1â€“5 stars) with reviews
- View ratings with JOIN queries (movie + user + rating)
- Update or delete ratings
- Tabular display using `tabulate` library

---

##  Technologies Used
- **Python** (core logic)
- **SQLite3** (database management)
- **Tabulate** (for clean tabular output)
- **Datetime** (for rating timestamps)
